# StyleSeed Design System

StyleSeed teaches AI how designers think — not just what brands look like.

## Core Components
- **69 Visual Design Rules** across 6 categories
- **48 React Components** (32 primitives + 16 dashboard patterns)
- **5 Motion Seeds**: Spring, Silk, Snap, Float, Pulse
- **Brand Skins**: Toss, Stripe, Linear, Vercel, Notion + 58 more

## When to Use
- Creating professional UI that avoids generic AI aesthetics
- Design token architecture (primitive → semantic → component)
- Motion design with personality-driven animations
- Brand-consistent design system generation

## Quick Reference
- Use `#2A2A2A` instead of `#000` for refined blacks
- Shadows at 4% opacity
- All content lives inside cards, not on page backgrounds
- Vary KPI card designs to avoid repetition
- All animations respect `prefers-reduced-motion`
