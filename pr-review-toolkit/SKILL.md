# PR Review Toolkit

Use these specialized agents for comprehensive PR reviews:
- **code-reviewer**: General code quality and best practices
- **silent-failure-hunter**: Find error handling gaps, swallowed errors, missing validation
- **code-simplifier**: Identify over-engineering and suggest simplifications
- **comment-analyzer**: Check if review comments were properly addressed
- **pr-test-analyzer**: Evaluate test coverage and quality
- **type-design-analyzer**: Review type system design and API boundaries
